# Task Manager

## Description 
A quick and simple way to manage tasks through the command line. Storing the users and tasks in text files. With functionality for
an admin to generate summaries of the tasks.

## Table of Contents
* [Description](#Description)
* [Installation](#Installation)
* [Usage](#Usage)
* [Credits](#Credits)

## Installation
1. Download the all the files.
2. Extract files to a folder.
3. Run the program using task_manager.py

## Usage 
1. Run the program
2. Sign in with your username and password or use Username: admin, Password: password to create a new user
3. View and manage tasks using the options listed in the program.
